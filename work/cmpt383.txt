Topic idea:
A web-based budget management system where users can record their income and expense.
Some calculations, including net-income, remaining monthly budget, expense percentage will be presented.

Languages:
JavaScript: function as client, interact with user, and send user input to Python server.
Python: function as server, directly communication with JavaScript.
C++: function like a calculator, do the calculation requested by Python server.

Inter-Languague communication:
JS - PYTHON : RabbitMQ
PYTHON - C++ : SWIG

Deployment Technology: Vagrant VM
